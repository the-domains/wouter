X explained

If you ever have been hacking away with a Linux desktop or even OS X (now macOS) chances are you have encountered X.

X, short for X11 is a framework for window servers. A window server is the magic behind all those fancy windows and graphics you see on your computer screen. Microsoft Windows and Apple macOS have their own propietary window servers. Most Linuxes and Unixes use X.org however (an implementation of X11). But how does this work? To learn more read this masterfully crafted series explaning X, it even runs an X11 server in javascript! What are you waiting for?
